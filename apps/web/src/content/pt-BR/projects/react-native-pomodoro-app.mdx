A mobile pomodoro app to help you manage your time and let you focus on any tasks.

## Motivation

I created this project to compare the development experience between React Native and Electron. I wanted to know how easy it is to create a desktop and mobile application using web technologies.

## Features

- [x] Start timer
- [x] Stop timer
- [x] Play sounds
- [x] Show notifications
