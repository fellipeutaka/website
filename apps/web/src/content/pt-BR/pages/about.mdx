## Quem sou eu

Eu sou um desenvolvedor Full-Stack brasileiro. Tenho <DisplayAge /> e crio aplicações para web, dispositivos móveis e desktop. Sou apaixonado por tecnologia e programação. Estou sempre em busca de novos desafios e aprendendo coisas novas. Atualmente estou trabalhando com TypeScript, React, Next.js, Node.js e outras tecnologias.

Meu primeiro contato com programação foi em 2014 (eu tinha 9 anos) quando tentei criar meu primeiro mod para [Minecraft](https://www.minecraft.net) usando Java. Depois tentei criar um jogo com Windows Form usando Visual Basic em 2017. Mas só comecei a estudar tecnologias web em 2020, quando iniciei meu curso de Desenvolvimento de Sistemas.

## Sobre este site

- Framework: [Next.js](https://nextjs.org)
- Linguagem: [TypeScript](https://www.typescriptlang.org)
- API: [tRPC](https://trpc.io)
- Banco de dados: [PostgreSQL](https://www.postgresql.org) fornecido por [Neon](https://neon.tech)
- ORM: [Drizzle](https://orm.drizzle.team)
- Autenticação: [NextAuth.js](https://authjs.dev)
- Deployment: [Vercel](https://vercel.com)
- Conteúdo: [MDX](https://mdxjs.com)
- Componentes: [shadcn/ui](https://ui.shadcn.com) e [Aceternity UI](https://ui.aceternity.com)
- Estilização: [Tailwindcss](https://tailwindcss.com)
- Code syntax highlighting: [Shiki](https://shiki.style)
