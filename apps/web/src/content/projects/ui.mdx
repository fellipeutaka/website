A Design System with accessible and customizable components powered by [shadcn/ui](https://ui.shadcn.com)

## Motivation

I created this project to help me build web applications faster and following the shadcn/ui components. So, I decided to create and publish these components to NPM to use in my projects.
The main goal of this project is to set up shadcn/ui faster and easier.

## How to use

1. Install the components package and the icons package

```bash
npm install @fellipeutaka/ui lucide-react
```

2. Install the utilities package

```bash
npm install @fellipeutaka/styles -D
```

3. Extend the Design System theme on your Tailwind CSS config file.

For ESM (`tailwind.config.js`, `tailwind.config.mjs` `tailwind.config.ts`)

```js
import { defineTailwindConfig } from "@fellipeutaka/styles";

export default defineTailwindConfig({
  content: ["./src/{app,screens,components}/**/*.{ts,tsx}"],
});
```

For CJS (`tailwind.config.js`, `tailwind.config.cjs`)

```js
const { defineTailwindConfig } = require("@fellipeutaka/styles");

module.exports = defineTailwindConfig({
  content: ["./src/{app,screens,components}/**/*.{ts,tsx}"],
});
```

4. Import `theme.css` file at your `globals.css`.

```css
/* default theme */
@import "@fellipeutaka/styles/theme.css";

@tailwind base;
@tailwind components;
@tailwind utilities;
```

5. Use the components

```jsx
import { Button } from "@fellipeutaka/ui/button";

export function App() {
  return <Button>Hello World!</Button>;
}
```
