A Discord bot that plays random "Vai dar namoro" sounds. "Vai dar namoro" (It's going to date) is a special segment presented and directed by Rodrigo Faro in the program Hora do Faro (a brazilian television program).

## How to use

1. Create a [Discord Bot](https://www.androidpolice.com/how-to-make-discord-bot)

2. Install [Git](https://git-scm.com) and [Node.js](https://nodejs.org) on your computer

3. Clone the repository

```bash
git clone https://github.com/fellipeutaka/cook.git
```

4. Install the dependencies

```bash
npm install
```

5. Create a `.env` file based on the `.env.example` file and insert your Discord Token and Discord Client ID

6. Start the development server

```bash
npm run dev
```
